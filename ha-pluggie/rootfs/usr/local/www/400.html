Configuration Required

Home Assistant needs additional configuration to work with Pluggie.

What happened?

Home Assistant rejected the connection because it doesn't recognize Pluggie as a trusted proxy. This is a security feature that prevents unauthorized access.

How to fix it

1Open your Home Assistant configuration.yaml file

2Add the following configuration:

http:
  use_x_forwarded_for: true
  trusted_proxies:
    - 172.30.32.0/23  # Pluggie addon network

3Restart Home Assistant

Go to Settings → System → Restart or use the command line.

4Try accessing your Home Assistant again through Pluggie
